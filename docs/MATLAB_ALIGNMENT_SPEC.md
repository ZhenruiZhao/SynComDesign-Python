# MATLAB Alignment Specification

MATLAB reference exports are generated by:

```matlab
cd('D:\SynComDesign-MATLAB-community-medium-fix')
addpath(genpath(pwd))
exportPythonReference
```

The exporter writes `python_reference_exports/` with resolved config, detected models, model validation, biomass reactions, all combinations, medium mapping, reaction classification, bounds tables, community build trace, objective trace, flux mapping, community summary, failed combinations, and a run log.

Python alignment command:

```bash
syncomdesign compare-matlab \
  --config config/syncomdesign_config.yml \
  --reference D:\SynComDesign-MATLAB-community-medium-fix\python_reference_exports \
  --outdir results_compare_matlab
```

Comparison tables:

- `models_detected`
- `all_combinations`
- `medium_mapping`
- `reaction_classification`
- `external_medium_bounds`
- `interface_bounds`
- `internal_transport_bounds`
- `community_summary`
- `flux_mapping`
- `objective_trace`

Numeric tolerance defaults to `1e-6`. Medium bounds and interface/internal bounds are priority checks. If total biomass is positive in one implementation and zero in the other, that is a severe mismatch.
